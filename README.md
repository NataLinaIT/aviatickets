
## Avia tickets search App

### Technologies used:
* HTML5, CSS3, Javascript
* axiox
* materialize-css
* webpack


You can run app:
### `npm run dev`

Runs the app in the development mode.
Open [http://localhost:9000](http://localhost:9000) to view it in the browser.
