
## Avia tickets search App

### Involved Technologies & Tools:
#### Development
* HTML5, CSS3, Javascript
* axiox
* materialize-css
* webpack
#### Tests
* jest
* cypress

You can run app:
##### `npm run dev`

Runs the app in the development mode.
Open [http://localhost:9000](http://localhost:9000) to view it in the browser.

Runs UNIT tests:
##### `npm run test` 

Runs E2E tests:
##### `npm run cypress:open`
